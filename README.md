# Création d'une branche

Faire des exercices dans [code wars](https://www.codewars.com/collections/php-beginner).

## Mise en place git

```bash
git branch [nom]
git checkout [nom]
mkdir [nom]
cd [nom]
```

## Rendu des exos
Créer un fichier `.php` par exercice et faites un push (noublier pas de pull avant).
